# Веб-визуализация Igus ReBeL (Foxglove Studio)

Этот метод позволяет управлять роботом через браузер, без установки X-сервера (VcXsrv).

## Запуск
1. Откройте терминал в папке `ROS2`.
2. Запустите скрипт:
   ```powershell
   docker-compose run --rm -p 9090:9090 ros2-dev bash -c "./start_web_sim.sh"
   ```
   *Примечание: Мы явно пробрасываем порт 9090, так как в режиме `network_mode: host` на Windows порты не всегда работают корректно без явного указания, либо если вы используете WSL2 backend.*

## Настройка Foxglove Studio
1. Откройте браузер (Chrome/Edge) и перейдите на [https://studio.foxglove.dev](https://studio.foxglove.dev).
2. Нажмите **Open connection**.
3. Выберите **Rosbridge**.
4. URL: `ws://localhost:9090`.
5. Нажмите **Open**.

## Настройка интерфейса
1. Добавьте панель **3D**.
2. В настройках панели (справа):
   - **Display Frame**: `world`
3. Вы должны увидеть модель робота.
4. Чтобы управлять роботом, добавьте панель **Teleop** или используйте скрипты.
   *Примечание: Для полноценного MoveIt планирования через веб может потребоваться дополнительная настройка плагинов Foxglove, но визуализация будет работать сразу.*

## Остановка
Нажмите `Ctrl+C` в терминале.
