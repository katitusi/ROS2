# ROS2 Project

Проект на ROS2 (Robot Operating System 2).

## Требования

- ROS2 (Humble/Iron/Jazzy)
- Python 3.8+
- colcon

## Установка

```bash
# Клонировать репозиторий
git clone <URL>

# Собрать пакеты
colcon build

# Активировать окружение
source install/setup.bash
```
